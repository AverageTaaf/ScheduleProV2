Help & Instructions

Getting Started

Welcome to SchedulePro! Here's how to use the application:

  1. Add a new task using the form at the top
  2. Set a due date, difficulty level, and importance
  3. Add checkpoints to break down your task
  4. Drag tasks between columns as you progress
  5. Click on a task to view or edit details

Features

  • Drag & Drop: Move tasks between columns by dragging
  • Search: Use the search bar to find specific tasks
  • Sorting: Sort tasks by date, importance, or progress
  • Filtering: Show all, active, or completed tasks
  • Statistics: View your task completion stats
  • Export/Import: Backup your tasks or move them to another device

Tips

  • Use the importance indicator to prioritize tasks
  • Break down complex tasks into multiple checkpoints
  • Regularly check the statistics to track your productivity
  • Use the dark mode toggle for comfortable nighttime use
